Now that we know the coeffients of the power series, we can write

\begin{align*} y \amp= a_0 + a_1 x + a_2 x^2 + a_3 x^3 + \ldots\\ \amp= a_0 + a_1 x - \frac{1}{2!} a_0 x^2 - \frac{1}{3!} a_1 x^3 + \ldots\\ \amp = a_0 ( 1 - \frac{1}{2!} x^2 + \frac{1}{4!} x^4 - \ldots) + a_1 (x - \frac{1}{3!} x^3 + \frac{1}{5!}x^5 - \ldots) \end{align*}

which allows us to write \(y = a_0 y_1 + a_0 y_2\) where

\begin{equation*} y_1(x) = \sum_{n=0}^\infty \frac{(-1)^n}{(2n)!} x^{2n}, \hspace{1cm} y_2 = \sum_{n=0}^\infty \frac{(-1)^n}{(2n+1)!} x^{2n+1}. \end{equation*}
